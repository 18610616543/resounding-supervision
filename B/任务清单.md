### 任务 B-2：saveConfig 增加必填字段校验

- **目标**：防止用户保存空名称或不合法配置
- **预计时间**：20分钟
- **具体步骤**：
  1. 在 `_doSaveConfig()` 函数（约 L1006）的 `name` 获取后增加校验：
     - 名称不能为空
     - 名称不能超过20字符
     - 名称不能为纯空格
  2. 使用 `Toast.show('请填写智能体名称', 'error')` 给出明确提示
  3. 校验不通过时 `return` 中断保存
- **验收标准**：输入空名称点保存 → 弹出红色错误提示，不执行保存

### 任务 B-3：修复边界 Bug（至少1个）

- **目标**：修复一个影响用户体验的 Bug
- **预计时间**：30分钟
- **候选 Bug**（选择1-2个）：
  1. **删除最后一个 Agent 后 UI 残留**：删完所有 agent 后，检查 `_renderAgents()` 是否正确显示空状态
  2. **编辑时切换性别预设被清空**：在 `setGender()`（L740）中切换性别会清空 `selectedPresets`，检查是否应弹出确认提示
  3. **快速双击保存按钮导致重复创建**：`_doSaveConfig()` 中按钮未及时 disabled，检查是否可能重复提交
- **具体步骤**：
  1. 选一个 Bug，复现问题
  2. 定位代码原因
  3. 编写修复代码
  4. 本地验证修复有效
- **验收标准**：Bug 不再复现，修复后无引入新问题
