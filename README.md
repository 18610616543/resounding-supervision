# resounding-supervision · 情感陪伴虚拟人

> 一个面向个人用户的情感陪伴 AI 虚拟人，提供智能体管理与情感聊天交互两大核心体验。
>
> 仓库地址：[https://github.com/18610616543/resounding-supervision](https://github.com/18610616543/resounding-supervision)

---

## 项目背景

在现代快节奏生活中，越来越多的人面临情感倾诉渠道缺失的问题。resounding-supervision 旨在打造一个可定制的情感陪伴 AI 虚拟人，用户可以根据个人偏好自由配置 AI 角色的性别、语言、语音和人格特征，并通过自然对话获得情感回应。项目采用纯前端技术栈，无需后端服务，数据完全存储在用户本地浏览器中，保障隐私安全。

---

## 核心功能

### 管理控制台（`index.html`）

- **智能体管理**：创建、编辑、删除、保存情感陪伴 AI 角色，每个智能体独立配置
- **人格配置双分支**：
  - 预设模式：10 种男女预设人格（温柔/活泼/专业/撒娇/幽默 × 男/女），支持多选自由融合
  - 自定义模式：自由编写系统提示词，支持 `{{assistant_name}}` 占位符
- **AI 提示词润色**：对自定义提示词进行 5 维度评分（身份定义/内容丰富度/语言风格/安全约束/结构清晰度），自动生成优化建议和润色后文本，支持一键应用
- **实时人格预览**：输入提示词后即时分析人格特征，生成特征标签、人格描述和模拟对话示例
- **分享码系统**：一键生成/导入人格配置分享码，内置社区推荐模板
- **用户认证**：注册、登录、记住密码，内置演示账号快速体验
- **数据备份**：一键导出/导入 JSON 格式全量备份

### 情感聊天交互（`resounding-supervision.html`）

- **多维情感识别**：基于关键词层级权重算法，对用户输入进行情感分类（开心/难过/惊讶/生气/温暖/安慰/鼓励/孤独/思念/感谢/日常等），并计算情感强度
- **智能对话回复**：按情感强度（轻度/中度/重度）分档匹配回复，支持上下文感知和情感趋势检测
- **语音交互**：Web Speech API 实现 TTS 语音合成朗读回复 + ASR 语音识别输入
- **角色动画系统**：情感驱动的表情切换（😊/😢/😲/🥰）、呼吸动画（正常/加速/缓慢）、抖动弹跳等多级动画反馈
- **主动陪伴模式**：渐进式无操作检测（30秒 → 1分钟 → 3分钟 → 8分钟），按时间段和用户兴趣标签生成智能话题主动推送

## 使用说明

### 管理控制台

1. 打开 `index.html`，首次访问需注册账号或使用演示账号登录
2. 点击右上角 **「+ 创建智能体」** 按钮
3. 填写智能体名称，选择性别、语言和语音角色
4. 进入 **「人格配置」** 标签页，选择预设人格（可多选）或切换到自定义模式编写提示词
5. 点击 **「AI 润色」** 获取提示词优化建议，满意后点 **「应用润色」**
6. 切换至 **「人格预览」** 查看即时人格分析结果
7. 点击 **「保存」** 完成创建，可在左侧列表管理所有智能体
8. 点击智能体卡片上的分享图标生成分享码，或在顶部导入他人的分享码

### 情感聊天交互

1. 打开 `resounding-supervision.html`，直接与 AI 角色对话
2. 输入表达情绪的内容（如"今天加班好累"），系统自动识别情感并匹配回复
3. 角色会根据当前情感自动切换表情和动画效果
4. 点击角色可触发弹跳互动动画
5. 点击 **🎤 麦克风** 按钮使用语音输入
6. 点击右上角开关开启 **主动陪伴模式**，空闲时自动发起话题
