# resounding-supervision · 情感陪伴机器人

> 面向个人用户的情感陪伴 AI 机器人，提供智能体管理与情感聊天交互两大核心体验。
>
> 仓库地址：[https://github.com/18610616543/resounding-supervision](https://github.com/18610616543/resounding-supervision)

---

## 核心功能

### 管理控制台（`index.html`）

- **智能体管理**：创建、编辑、删除、保存 AI 角色。每个智能体可独立配置性别、语言、语音角色和人格
- **人格配置双分支**：
  - 预设模式：10 种男女预设人格（温柔/活泼/专业/撒娇/幽默 × 男女），支持多选自由融合
  - 自定义模式：自由编写系统提示词，支持 `{{assistant_name}}` 占位符
- **AI 提示词润色**：对自定义提示词进行 5 维度评分（身份定义/内容丰富度/语言风格/安全约束/结构清晰度），自动生成优化建议和一键应用润色
- **实时人格预览**：输入提示词后即时分析人格特征，生成特征标签、人格描述和模拟对话示例
- **分享码系统**：一键生成/导入人格配置分享码，内置社区推荐码
- **用户认证**：注册、登录、记住密码，演示账号 `demo@resounding-supervision.ai`
- **数据备份**：一键导出/导入 JSON 格式全量备份

### 情感聊天交互（`resounding-supervision.html`）

- **多维情感识别**：基于关键词层级权重算法，对用户输入进行情感分类（开心/难过/惊讶/生气/温暖等），并计算情感强度
- **智能对话回复**：按情感强度（轻度/中度/重度）分档匹配回复，支持上下文感知和情感趋势检测
- **语音交互**：Web Speech API 实现 TTS 语音合成朗读回复 + ASR 语音识别输入
- **角色动画系统**：情感驱动的表情切换（😊/😢/😲/🥰）、呼吸动画（正常/加速/缓慢）、抖动弹跳等多级动画反馈
- **主动陪伴模式**：渐进式无操作检测（30秒 → 1分钟 → 3分钟 → 8分钟），按时间段和用户兴趣标签生成智能话题主动推送

## 快速开始

```bash
# 1. 克隆仓库
git clone https://github.com/18610616543/resounding-supervision.git

# 2. 进入目录
cd resounding-supervision

# 3. 用浏览器打开管理控制台
start index.html

# 或打开情感聊天页面
start resounding-supervision.html
```

> 无需安装任何依赖，无需构建步骤，直接双击 HTML 文件即可运行。推荐使用 Chrome 或 Edge 浏览器。

## 使用说明

### 控制台

1. 打开 `index.html`，点击右上角 **+ 创建智能体**
2. 填写智能体名称，选择性别和语言
3. 在人格配置中选择预设模式（多选组合）或自定义模式（编写提示词）
4. 点击 **AI 润色** 可获得提示词优化建议
5. 左侧可实时预览人格效果
6. 保存后可通过分享码分享给他人

### 情感聊天

1. 打开 `resounding-supervision.html`，直接与 AI 角色对话
2. 输入表达情绪的语句（如"今天好开心"、"有点难过"），观察角色表情变化
3. 点击右上角开关可开启 **主动陪伴模式**
4. 点击角色可触发弹跳动画
5. 支持语音输入（点击🎤）和语音朗读
